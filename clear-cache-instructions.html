🔧 브라우저 캐시 및 Service Worker 완전 초기화

⚠️ 문제 상황: Service Worker 캐시 충돌로 인한 리소스 로딩 실패

📱 Chrome/Safari 완전 초기화 방법

1단계: 개발자 도구 열기

F12 또는 Cmd+Option+I (Mac) / Ctrl+Shift+I (Windows)

2단계: Application 탭으로 이동

개발자 도구에서 "Application" 탭 클릭

3단계: Storage 완전 삭제

왼쪽 메뉴에서:

  • Service Workers → 등록된 SW 모두 "Unregister" 클릭
  • Cache Storage → 모든 캐시 삭제
  • Local Storage → 모든 데이터 삭제
  • Session Storage → 모든 데이터 삭제

4단계: 하드 리프레시

Cmd+Shift+R (Mac) / Ctrl+Shift+R (Windows)

또는 새로고침 버튼 우클릭 → "하드 새로고침" 선택

🚀 빠른 해결 방법

시크릿/프라이빗 브라우징 모드 사용

Cmd+Shift+N (Mac) / Ctrl+Shift+N (Windows)

새 시크릿 창에서 http://localhost:3000/dashboard/admin/documents 접속

✅ 최종 확인 단계

1. 콘솔에서 Service Worker 에러가 사라졌는지 확인

2. 새로운 4개 탭 인터페이스가 정상 로딩되는지 확인

3. 각 탭 클릭 시 정상 전환되는지 확인
