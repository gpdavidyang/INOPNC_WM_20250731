🔧 Service Worker 초기화 도구

Service Worker 캐시 문제를 해결하기 위한 도구입니다.

📋 사용 방법

  1. 브라우저에서 이 파일을 엽니다: file:///Users/davidyang/workspace/INOPNC_WM_20250731/clear-service-worker.html
  2. "Service Worker 초기화" 버튼을 클릭합니다
  3. "모든 캐시 삭제" 버튼을 클릭합니다
  4. 브라우저를 완전히 종료했다가 다시 엽니다
  5. "앱으로 이동" 버튼을 클릭하거나 직접 http://localhost:3000 으로 접속합니다

브라우저 개발자 도구에서 직접 실행:

// Service Worker 제거
navigator.serviceWorker.getRegistrations().then(function(registrations) {
    for(let registration of registrations) {
        registration.unregister();
    }
});

// 모든 캐시 삭제
caches.keys().then(function(names) {
    for (let name of names) {
        caches.delete(name);
    }
});
