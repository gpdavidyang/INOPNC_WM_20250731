모달 중앙 정렬 테스트

수정 내용:

  • position: fixed 추가
  • top: 50%, left: 50% 설정
  • transform: translate(-50%, -50%) 로 정확한 중앙 정렬
  • width: 90vw 로 반응형 너비 설정
  • overflow: hidden 후 flex layout 사용
  • 헤더 고정, 콘텐츠만 스크롤 가능
