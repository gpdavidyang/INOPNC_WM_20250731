2. 브라우저 콘솔 테스트 명령어

브라우저 개발자 도구 콘솔에서 실행할 수 있는 테스트 명령어입니다:

// 1. 직접 등록 버튼 클릭 시뮬레이션 document.querySelector('button[onClick*="setIsUploadModalOpen"]')?.click(); // 2. 검색 필드에 텍스트 입력 const searchInput = document.querySelector('input[placeholder*="서류명"]'); if (searchInput) { searchInput.value = '안전교육'; searchInput.dispatchEvent(new Event('change', { bubbles: true })); } // 3. 상태 필터 변경 const statusSelect = document.querySelectorAll('select')[0]; if (statusSelect) { statusSelect.value = 'pending'; statusSelect.dispatchEvent(new Event('change', { bubbles: true })); } // 4. 모달 상태 확인 console.log('업로드 모달 열림:', !!document.querySelector('[role="dialog"]'));

3. 테스트 시나리오

시나리오 1: 서류 업로드

  1. 직접 등록 버튼 클릭
  2. 제출자 선택: worker@inopnc.com
  3. 서류 유형 선택: 안전교육이수증
  4. 파일 드래그 앤 드롭 또는 클릭하여 선택
  5. 서류 등록 버튼 클릭
UI 테스트 가능

시나리오 2: 서류 검색 및 필터링

  1. 검색창에 "안전" 입력
  2. 상태 필터를 "검토중"으로 변경
  3. 서류 유형을 "안전교육이수증"으로 선택
  4. 새로고침 버튼 클릭
UI 테스트 가능

시나리오 3: 서류 상세보기 및 승인

  1. 목록에서 상세보기 버튼(눈 아이콘) 클릭
  2. 검토 노트 편집 버튼 클릭
  3. 검토 내용 입력
  4. 승인 또는 거부 버튼 클릭
데이터베이스 마이그레이션 필요
